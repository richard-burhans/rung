# Code of Conduct

`rung` adopts the [Galaxy Community Code of Conduct](https://galaxyproject.org/community/coc/).

We want this project to be a welcoming, inclusive, respectful, and collaborative place — for
people writing their first scraper as much as for people who have run fleets of them. Please read
the Galaxy Code of Conduct in full; it governs every space this project uses, including the
GitHub repository, its issues and pull requests, and any discussion that references the project.

## In short

- Be welcoming and patient, especially with newcomers.
- Be respectful and considerate; critique ideas and code, never people.
- Be collaborative — assume good faith, and prefer explaining to gatekeeping.
- Accept responsibility, apologize to those affected by our mistakes, and learn from them.

Harassment and exclusionary behavior are not acceptable, in any form.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, please report it to the
project maintainer at **richard.burhans@gmail.com**. Reports will be handled discreetly and in
good faith, and the reporter's identity will not be shared without their consent.

Where a report concerns the maintainer, or where you would prefer to escalate outside this
project, the Galaxy Code of Conduct describes its own reporting and enforcement channels; those
routes are available to you and we will cooperate with them.

## Scope

This Code of Conduct applies within all project spaces, and also applies when an individual is
officially representing the project in public spaces.
